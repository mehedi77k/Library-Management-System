# Dependencies

This project is a Java Spring Boot application. Dependencies are managed with Maven in [pom.xml](pom.xml) and use the Spring Boot 3.5.0 BOM for versions.

## Runtime Dependencies
- Spring Boot Starter Web (version managed by Spring Boot 3.5.0)
- Spring Boot Starter Thymeleaf (version managed by Spring Boot 3.5.0)
- Thymeleaf Extras Spring Security 6 (version managed by Spring Boot 3.5.0)
- Spring Boot Starter Data JPA (Hibernate) (version managed by Spring Boot 3.5.0)
- Spring Boot Starter Validation (version managed by Spring Boot 3.5.0)
- Spring Boot Starter Security (version managed by Spring Boot 3.5.0)
- MySQL Connector/J (version managed by Spring Boot 3.5.0)
- Spring Boot DevTools (optional, runtime) (version managed by Spring Boot 3.5.0)

## Test Dependencies
- Spring Boot Starter Test (version managed by Spring Boot 3.5.0)
- Spring Security Test (version managed by Spring Boot 3.5.0)

## Java Version
- Java 21

## Build Tool
- Maven 3.8+
