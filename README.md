A simple **Library Management System** built using **Java**, designed to manage books and basic library operations without using any external database.

---

# 📱 Overview

This **Library Management System** is a beginner-friendly Java project that demonstrates how to manage library operations using core programming concepts.

Unlike advanced systems, this project **does NOT use any database**.  
All data is handled using **Java classes, objects, and in-memory structures**.

The system focuses on implementing:

- Object-Oriented Programming (OOP)
- Basic CRUD operations
- Console-based interaction

# 🧠 How the System Works

1. Program starts with a menu.
2. User selects an option (Add / View / Issue / Return).
3. Books are stored in memory using objects.
4. When issuing a book:
   - System checks availability
5. When returning:
   - Status is updated
6. Data is not saved permanently (resets after program exit)

---

# 📊 Data Handling

This system does NOT use database.

Instead, it uses:

- Arrays / ArrayList
- Object instances
- Runtime memory storage

⚠️ Data will be lost after program closes.
